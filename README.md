# 動画ダウンローダー (Video Downloader)

動画のURLを入力するだけで簡単にダウンロードできるWebツールです。

## 🌐 デモ

GitHub Pagesでホストされています：
`https://[your-username].github.io/cdnDL/`

## ✨ 機能

- 🎥 動画URLから直接ダウンロード
- 📱 レスポンシブデザイン（モバイル対応）
- 🇯🇵 日本語インターフェース
- ⚡ クライアントサイドで動作（サーバー不要）

## 🚀 使い方

1. 動画のURLを入力欄に貼り付ける
2. 「ダウンロード」ボタンをクリック
3. ダウンロードが開始されます

### 対応形式

- MP4形式の動画
- 直接リンクの動画ファイル
- 例: `https://cdn.videy.co/o4UO4CHD1.mp4`

## 🛠️ GitHub Pagesでのデプロイ

1. リポジトリの設定（Settings）を開く
2. 「Pages」セクションに移動
3. Source（ソース）で「Deploy from a branch」を選択
4. Branch（ブランチ）で「main」（または使用しているブランチ）を選択
5. フォルダは「/ (root)」を選択
6. 「Save」をクリック

数分後、`https://[your-username].github.io/cdnDL/` でアクセスできるようになります。

## 📝 注意事項

⚠️ **著作権について**

著作権で保護されたコンテンツのダウンロードは違法です。
自分が権利を持つ動画、または許可された動画のみダウンロードしてください。
